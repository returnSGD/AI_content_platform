QUESTION 1

做什么内容？

个性化爬虫 + RAG混合检索 + LLM-Rec重排序，每天凌晨为你生成Top-5选题推荐，附带差异化切入角度。

QUESTION 2

怎么做？

垂直领域LoRA微调 + 五步生成Pipeline + 风格池防同质化，从大纲到多平台适配，每步可编辑。

三层闭环架构

发现 → 创作 → 归因，数据飞轮持续自进化

Layer 1

内容机会发现

  • 个性化嗅探爬虫（三梯队数据源）
  • BGE向量化 + Milvus入库
  • Hybrid检索：Dense + Sparse + RRF融合
  • LLM-Rec重排序 + 差异化角度
  • XGBoost粉丝-内容匹配分类
Layer 2

AI 辅助创作

  • 5领域LoRA微调（r=16, rank=16）
  • 五步生成Pipeline（每步可编辑）
  • 风格池采样 · 防同质化三层防御
  • 品牌专属LoRA（B端核心能力）
  • 小红书/抖音/B站一键适配
Layer 3

归因反馈

  • 全链路数据追踪（平台API+埋点）
  • 贝叶斯反演归因（PyMC/NumPyro）
  • 转化贡献度 + 95% HDI可信区间
  • LLM周报 + 三级预警
  • 反馈回传Layer1和Layer2 → 自进化
选题+角度 → Layer 2
发布内容 → Layer 3
⟲ Layer 3 归因结果回传 Layer 1（推荐权重更新） & Layer 2（训练集更新）
