# ContentEngine AI — AI 驱动的内容策略引擎

AI 早期创业项目 · 轻量机会验证 · 宠物 AI 方向 → **内容种草赛道**

## 一句话定位

帮中小商家和创作者回答三个问题：**做什么内容？怎么做？效果如何？** —— 并让这三个答案形成自进化的数据飞轮。

## 项目背景

本项目为 AI 创业竞赛的完整机会验证方案。选题方向为"宠物 AI"，研究过程中发现：**养宠人的核心 AI 需求已被大量产品覆盖（智能喂食器、健康监测、在线问诊等），但为这些宠物品牌/商家提供内容策略和营销工具的 AI 产品仍属蓝海**。因此将方向锚定为"内容策略引擎"—— 服务宠物赛道的品牌方和创作者，同时具备向美妆、食品、母婴等高频种草品类横向扩展的能力。

## 产品架构

```
┌──────────────────────────────────────────────┐
│                三层闭环架构                      │
│                                                │
│  Layer 1: 内容机会发现                           │
│  爬虫 → Embedding → 混合检索 → LLM-Rec排序       │
│  回答：粉丝想看什么？什么内容还有红利？              │
│                      ↓                         │
│  Layer 2: AI 辅助创作                            │
│  LoRA微调 → 五步Pipeline → 风格池防同质化          │
│  回答：怎么写？怎么写得比竞品好？                    │
│                      ↓                         │
│  Layer 3: 归因反馈                               │
│  全链路追踪 → 贝叶斯归因 → LLM周报 → 三级预警       │
│  回答：效果如何？下次怎么调整？                      │
│                      ↓                         │
│  反馈信号回传 Layer 1 & Layer 2 → 自进化飞轮       │
└──────────────────────────────────────────────┘
```

## 核心 Prompt 模板（10 个）

详见 `AI工作流与Prompt流程.md`：

| # | Prompt | 层级 | 调用时机 |
|---|--------|------|---------|
| 1 | 内容质量评分 | L1 | 离线批处理 |
| 2 | LLM-Rec 选题重排序 | L1 | 每日凌晨 |
| 3 | 粉丝画像生成 | L1 | 用户首次接入 |
| 4 | 大纲生成 | L2 Step1 | 用户触发 |
| 5 | 分段扩写 | L2 Step2 | 用户触发 |
| 6 | 风格润色 | L2 Step3 | 用户触发 |
| 7 | 质量自检 | L2 Step4 | 用户触发 |
| 8 | 多平台适配(小红书/抖音/B站) | L2 Step5 | 用户触发 |
| 9 | 周报生成 | L3 | 每周一08:00 |
| 10 | 品牌语音指南提取 | 跨层 | B端品牌首次接入 |

**设计原则**：角色锚定 → 信源锁定 → 结构化输出(强制JSON) → 不确定性标注 → 风格池注入 → 多层防御

## 商业模型

| 层级 | 用户 | 价格 | 核心功能 |
|------|------|------|---------|
| C端 Free | 个体创作者 | 0元 | 每日3次生成、基础选题 |
| C端 Pro | KOC | 39元/月 | 无限生成、3平台适配、7天追踪 |
| C端 Pro+ | 腰部KOL | 99元/月 | 竞品分析、高级风格模型、30天追踪 |
| B端 初创 | 小品牌 | 999元/月 | 3人协作、月100条、基础归因 |
| B端 成长 | 成长品牌 | 2999元/月 | 10人协作、月500条、品牌LoRA训练 |
| B端 企业 | 头部品牌 | 定制 | 私有化部署、API接入、定制模型 |

**获客路径**：内容驱动（自有产品即最佳案例）→ 社群渗透（平台合作）→ 标杆裂变

## 2周验证计划

| 假设 | 验证方法 | Go 条件 |
|------|---------|--------|
| H1: 策略决策是刚需痛点 | 15人深度访谈 + 诊断Demo | ≥60%自发表达策略焦虑 |
| H2: 愿为三合一付费 | Fake Door + 预售 + WOZ | 预售≥30%或Fake Door >2x均值 |
| H3: 归因是核心驱动力 | 访谈追问归因现状 | ≥50%认为归因"很需要" |

详见 `两周验证.txt`。

- 不做实时推荐（V2）
- 不做全平台（首批仅小红书+抖音）
- 不做自研大模型
- 不做视频/图像直接生成
- 不做全自动发布
- 不做跨用户数据混合
- 不做社交功能
